== Description ==

**Advanced Background Block** is a custom Gutenberg Block developed with **Gutenberg Native Components**. It allows you to design **Responsive Row with different backgrounds in Gutenberg Editor**. It comes with a lot of features so that you can easily make an attractive section.

### Overview
[youtube https://www.youtube.com/watch?v=PRNaUJe-1mE]

### Key Features
* Simple, light-weight, and super-fast
* Built with **Gutenberg Native Components**
* Design **Responsive Row or Section**
* Custom Padding on different devices
* Custom Margin on different devices
* **Different types of background** e.g color, gradient, image & video background
* Set Maximum Content Width
* **Enable Overlay**

== Installation ==

This section describes how to install the plugin and get it working.
e.g.
1. Upload the plugin files to the `/wp-content/plugins/section-builder-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Simply go to Gutenberg Block Editor and you will find a new category with a new Block named "Section Builder Block"
4. Drag and drop to use it.
